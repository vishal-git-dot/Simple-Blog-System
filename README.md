A lightweight and beginner-friendly **Blog Management System** built
using **PHP, MySQL, HTML, CSS, and JavaScript**.

------------------------------------------------------------------------

## 🚀 Features

### ✍️ Blog Post Management

-   Create new blog posts
-   Edit existing posts
-   Delete posts
-   View all posts
-   View single post in detail

### 📢 Publishing Control

-   Publish / Unpublish posts
-   Draft system for incomplete posts

### 💬 Comment System (Bonus)

-   Add comments to posts
-   View all comments per post

------------------------------------------------------------------------

## 🛠 Tech Stack

-   PHP
-   MySQL
-   HTML5
-   CSS3
-   JavaScript

------------------------------------------------------------------------

## ⚙️ Installation

1.  Clone repo
2.  Move to htdocs
3.  Create database: blog_system
4.  Import tables
5.  Run: http://localhost/Simple-Blog-System/
